1 dojo初步

1.1 结构

  • 导入dojo库
  • 运行在服务器上
  • 页面html代码
    • css 
      <style type="text/css">
@import "dojo目录/themes/样式文件名称"
@import "dojo目录/themes/dojo.css"
</style>
    • dojo.js的导入 
    <script type="text/javascript" src="dojo目录/dojo.dojo.js" djConfig="parseOnLoad:true"></script>

1.2 小部件的添加

  • 脚本添加 
<script type="text/javacript"> 
  dojo.require("dojo.form...dojo中对应的包名和部件名");
</script>
  • html代码(比如按钮) 
<button data-dojo-type="dijit.form.Button" id="按钮的id">按钮上显示的名称</button>

1.3 小部件与事件的关联

  • 脚本标签 
 <button do-jo-type="dijit.form.Button" id="按钮的id">
 按钮的名称
 <script type="dojo.method" event="onClick">
   alert("You pressed the button!");
 </script>
</button>

1.4 获取服务器数据

  • 回调函数处理数据
  • 按钮触发请求
  • 例子 
<script>
  function 成功方法();
  function 失败方法(); 

  dojo.xhrGet({
   url: '访问地址',
   load: 成功方法,
   error:失败方法
 });
</script>

1.5 get方法传送数据

  • 参数的传递 
dojo.xhrGet({
  url: "服务器地址",
  load: 成功方法,
  error: 失败方法,
  content:{参数名称:参数值}
});

1.6 通过id获取表单元素值的方法

  • dojo.byId(id名称).value;

1.7 post方法提交

  • 提交方法 xhrPost()
  • 参数为form属性
  • 例子 
dojo.xhrPost({
 url: "服务器地址",
 load:成功方法,
 error:失败方法,
 form:"formid"
});

2.2 属性 方法 扩展点

2.2.1 属性

  • 是用来控制显示或行为的数据元素
  • 只能在创建的时候设置一次

2.2.2 方法

  • 控制某些事情的函数

  • 可重构的函数

2.2.4 例子

 <div data-dojo-type="dijit.form.ValidatingTextBox" isValid="my.form.isValid"...>
    <script type="dojo/method" event="isValid">
     扩展的方法
   </script>
 <div>
 <!-- 通用方法 -->
my.form.isValid = function(isFocused){
   扩展方法 
}

3.1.6 dojo.forEach

  • 遍历
  • 例子 
dojo.query("select").forEach(
  function(selectTag) {
      selectTag.disabled = true;
  }
);
<!-- 简写 -->
dojo.query("select", document).forEach("item.disabled = true;");
