1.4 android 的应用结构分析

1.4.1 创建一个Android应用

  • create project
    • Android create project -n HellowWorld -t 6 -p HellowWorld -k org.crazyit.HellowWorld -a HellowWorld
  • 各个文件夹
    • res 各种资源文件
    • src java源文件
    • bin 生成的目标文件
    • gen 自动生成的R.java清单文件

1.4.2 自动生成的R.java

  • 每个资源对应R类的一个内部类
  • 每个具体的资源项对应于内部类的一个public static final int 类型的Field

1.4.3 res目录说明

  • 不同的资源放在不同的文件夹中
  1. 在java代码中使用资源
    • 资源项对应的field
  2. 在xml文件中使用资源
    • @<资源对应的内部类的类名>/<资源项的名称>
    • @+i/<标识符代号>

1.4.4 Android 应用的清单文件 AndroidMainifest.xml

  • 应用程序的包名
  • 包含的组件
  • 兼容的最低版本
  • 使用系统所需的权限声明
  • 其他程序访问程序所需的权限声明
